# Customer Segmentation using K-Means Clustering

## Task 2 - Machine Learning Internship

### Objective

To implement the K-Means Clustering algorithm to group customers of a retail store based on their purchase behavior and spending patterns.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

### Dataset

The dataset contains customer information such as:

* Annual Income
* Spending Score
* Customer ID

### Methodology

1. Load the dataset
2. Perform exploratory data analysis
3. Select relevant features
4. Use the Elbow Method to determine the optimal number of clusters
5. Apply K-Means Clustering
6. Visualize customer segments
7. Analyze cluster distribution

### Results

The K-Means algorithm successfully grouped customers into distinct clusters based on their income and spending behavior. These clusters can help businesses understand customer segments and develop targeted marketing strategies.

### Visualizations

* Elbow Method Graph
* Customer Segmentation Scatter Plot
* Cluster Distribution Chart

### Conclusion

This project demonstrates how unsupervised machine learning techniques such as K-Means Clustering can be used to identify meaningful customer segments from retail data.
